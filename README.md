# go-twiter-bot
Creating a twiter bot using GO


Libs:

github.com/dghubble/go-twitter
github.com/dghubble/oauth1

Configuração:

    Exportar variáveis de ambiente:

    CONSUMER_KEY
    CONSUMER_SECRET_KEY
    TOKEN_KEY
    TOKEN_SECRET_KEY

    Criar o nosso client

Funcionalidades:

    Realizar tweet
    Listar tweets por query
    Retweet de algum tweet
